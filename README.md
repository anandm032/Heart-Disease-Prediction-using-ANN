# ❤️ Heart Disease Prediction using ANN

## 📌 Overview

This project predicts whether a patient has heart disease using an **Artificial Neural Network (ANN)** built with **TensorFlow/Keras**.

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* TensorFlow/Keras
* Matplotlib

## 📂 Dataset

* Heart Disease Dataset
* 1025 Records
* 13 Input Features
* Target:

  * 0 = No Heart Disease
  * 1 = Heart Disease

## 🧠 Model

* Hidden Layer: 16 Neurons (ReLU)
* Output Layer: 1 Neuron (Sigmoid)
* Optimizer: Adam
* Loss Function: Binary Cross-Entropy
* Epochs: 50
* Batch Size: 32

## 🚀 Workflow

* Load Dataset
* Data Preprocessing
* Feature Scaling
* Train-Test Split
* Build ANN
* Train Model
* Evaluate Model
* Make Predictions

## 📊 Evaluation

* Accuracy
* Loss
* Confusion Matrix

## ▶️ Run

Install dependencies:

```bash
pip install tensorflow pandas numpy matplotlib scikit-learn
```

Run the project:

```bash
python ann_heart.py
```
